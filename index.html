Puzzles Instruction

You can solve the puzzle by dragging the puzzle pieces in correct order. To see a finished version of the puzzle click on the completed puzzle button. To set the puzzle to it's original order click on the reset button. Choose another puzzle option using left hand side 'Select puzzle here.'

Card's Extra Features

  • Stickers are draggable to canvas and on canvas.
  • There is a bin on canvas to delete stickers and recipient name. Drag sticker / recipient name to the bin to delete.
  • You can change the font of message and recepient name on canvas.
  • You can save 3 different copies of card at the bottom.
    Restiction: More than 3 copies are not allowed. I need to make improvement for this part and make it more functional for example any of 3 versions can be selected for printing. I may try this as part of final project.
  • Empty card are not allowed to print. You need to have atleast one sticker, a message and recepient name.
  • Use of tool tip to guide.

Project Features

  • Animated Logo
  • This is a single page website created using jquery, jquery ui, HTML, CSS.
  • This website is for the kids / parents and teachers.
  • Responsive Layout
  • Use of tool tips to guide.
  • Use of Modals.
  • Use of Jquery UI for sortable puzzle
  • Two .js files 1) For puzzles and learn table 2) For Cards
  • Two .css files 1) style.css (puzzles and learn table) 2) features.css (cards)
  • One .html file : index.html

Key Parts of Project

  • Menu: Handling single page layout. Showing and hiding divs according to menu. Highlight the active page on menu.
  • Sortable Puzzle
  • You can sort puzzle and see the movement of Sorting at the bottom. Result of the puzzle will be displayed once user will complete the puzzle.
  • You can see finished version of puzzles. User can close the finished version either clicking on close button or on 'X' of completed puzzle image. There is option to reset the puzzle.
  • Learn times Tables. Only numbers are allowed, you can not print empty table. It will throw an error. There is an option to reset the table.
  • Stickers are draggable in cards section.
  • User can delete stickers and receient name on canvas by dragging it to the bin (at the bottom of canvas)
  • User can change the font of canvas text.
  • This card is full of options, if you want to go back to any point. You don't need to click 'start over' button forcefully.
  • User is resticted by errors. You can not print empty canvas.
  • You can not save more than 3 cards.
